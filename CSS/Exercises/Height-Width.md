# **CSS Height/Width**

**1. Set the height of the h1 element to "100px".**

```
<style>
    h1 {
        height:100px;
    }
</style>

<body>
    <h1>This is a heading</h1>
    <p>This is a paragraph</p>
    <p>This is a paragraph</p>
</body>
```

**2. Set the width of the h1 element to "50%".**

```
<style>
    h1 {
        width:50%;
    }
</style>

<body>
    <h1>This is a heading</h1>
    <p>This is a paragraph</p>
    <p>This is a paragraph</p>
</body>
```
